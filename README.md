# Bijan Mosavat – Personal Website

This repository contains the source code for my personal website hosted with GitHub Pages.

## Live Website

https://bijanmosavat.github.io

## Overview

This website presents my professional profile as an Infrastructure & Platform Engineer.  
It includes:

- Profile summary
- Core technical capabilities
- Infrastructure and platform expertise
- Education and certifications
- Contact information
- Downloadable CV

The site is intentionally minimal, fast-loading, and dependency-free.

## Tech Stack

- HTML5
- CSS3
- GitHub Pages (static hosting)

No frameworks, no external libraries, no build tools.

## Deployment

The site is deployed using GitHub Pages from the `main` branch.

Repository name must be:

Deployment settings:
- Source: Deploy from branch
- Branch: main
- Folder: / (root)

After pushing changes to `main`, updates are automatically published.

## Customization

To modify the website:

1. Edit `index.html`
2. Commit changes
3. Push to the `main` branch
4. Wait 1–2 minutes for deployment
